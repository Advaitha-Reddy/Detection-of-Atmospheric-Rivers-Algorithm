Instructions for AMSR Data Download and TPW Plotting
-----------------------------------------------------------
This zip file contains the Python script for:
Downloading AMSR2 Total Precipitable Water (TPW) data.
Plotting the TPW data for a specific date.

---------------------------------------------------------
----------------------------------------------------------

1. Set the Output Directory for Data:
   ----------------------------------
   In the script, locate the following line:

       output_dir = 'your_download_directory'

   Replace 'your_download_directory' with the full path to the folder 
   where you want the downloaded AMSR .he5 files to be saved.

2. Set the Directory to Save Plots:
   --------------------------------
   Locate the following line:

       save_dir = 'your_plot_directory'

   Replace 'your_plot_directory' with the path where you want to save 
   the TPW plot images generated by the script.

3. Running the Script:
   -------------------
   When you run the script, it will prompt:

       Enter the reference date (YYYY-MM-DD):

   - If AMSR data is available for the given date or any of the 
     previous 30 days, it will download the files and generate a plot.
   - If no data is found in the last 30 days, the script will notify you.
